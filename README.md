
## 📌 Project Overview

This dashboard was built to analyze and monitor the organization's human capital across multiple dimensions — from headcount and demographics to performance and attrition. It consolidates HR data into a single, intuitive view that helps stakeholders quickly identify trends, risks, and opportunities within the workforce.

## 🎯 Key Questions Answered

- What is the current headcount and how is it distributed across departments?
- What are the attrition rates and what factors are driving employee turnover?
- How are employees distributed by gender, age group, and education level?
- What is the overall performance rating distribution across the organization?
- Which departments have the highest hiring and resignation activity?

---

## 📊 Dashboard Features

| Feature | Description |
|---|---|
| **KPI Cards** | Total Employees, Active Headcount, Attrition Rate, Average Tenure |
| **Demographics Breakdown** | Gender distribution, Age groups, Education levels |
| **Department Analysis** | Headcount and attrition by department |
| **Performance Overview** | Rating distribution across employee segments |
| **Filters / Slicers** | Interactive filtering by Department, Gender, Job Role |

## 🛠️ Tools & Technologies

- **Power BI Desktop** — Dashboard design and data modeling
- **Power Query** — Data cleaning and transformation
